# SHA-256 Hash Brute Force Demo

## Description
This project simulates a dictionary-based brute force attack on a hashed password using SHA-256.

## Concepts Demonstrated
- Password hashing
- Dictionary attacks
- Time measurement
- Attack simulation in controlled environment

## Limitations
- No salting
- No rate limiting
- Demonstration only

## Disclaimer
For educational use in controlled lab environments only.
